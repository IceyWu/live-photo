# 国际化（i18n）

## 内置语言包

目前内置两种语言：

| locale | 语言 |
|--------|------|
| `zh-CN`（默认） | 简体中文 |
| `en` | English |

```js
new LivePhotoViewer({ locale: 'en', ... });
```

## 覆盖单个文案

不需要切换整个语言包，只覆盖需要的字段：

```js
new LivePhotoViewer({
  labels: {
    live: 'LIVE',
    enableAutoplay: 'Autoplay On',
    disableAutoplay: 'Autoplay Off',
    mute: 'Mute',
    unmute: 'Sound On',
  },
  ...
});
```

## 完整 LivePhotoLabels 接口

```ts
interface LivePhotoLabels {
  live: string;            // 徽标文字，默认 "实况" / "LIVE"
  enableAutoplay: string;  // 菜单项：开启自动播放
  disableAutoplay: string; // 菜单项：关闭自动播放
  mute: string;            // 菜单项：静音
  unmute: string;          // 菜单项：开启声音
}
```

## 注册自定义语言

```js
import { LOCALE_LABELS } from 'live-photo';

LOCALE_LABELS['ja'] = {
  live: 'ライブ',
  enableAutoplay: '自動再生をオン',
  disableAutoplay: '自動再生をオフ',
  mute: 'ミュート',
  unmute: 'ミュート解除',
};

new LivePhotoViewer({ locale: 'ja', ... });
```

## 优先级

用户传入的值优先级从低到高：

1. 内置默认语言（`zh-CN`）
2. `locale` 指定的语言包
3. `labels` 字段（单项覆盖，最高优先级）
